# Zako的神奇抽卡工具

一个用于随机抽取 AI 绘画 Tag 的桌面工具，支持多种类别和画师串组合。

## 功能特点

- 🎴 **多类别抽卡** - 内置 20+ 服装/角色类别（女仆、巫女、精灵、魔法少女、哥特萝莉等）
- 🎨 **画师串支持** - 提供预设画师串、随机画师组合、自定义导入三种模式
- 📁 **自定义导入** - 支持导入自己的 Tag 类别和画师串 JSON 文件
- 🔒 **分级筛选** - G（普通）/ S（敏感）/ Q（可疑）/ E（露骨）四个级别可选
- 🔄 **固定模式** - 锁定当前 Tag，仅切换画师串
- 📋 **一键复制** - 抽取结果可直接复制到 AI 绘画工具中使用

## 使用方法

前往 [Release](../../releases) 下载 `Zako的神奇抽卡工具.exe`，双击运行即可。

## JSON 格式说明

如需导入自定义 Tag，JSON 格式如下：
```json
{
  "英文tag内容": "编号1",
  "英文tag内容2": "编号2"
}
```
